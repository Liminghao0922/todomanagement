9️⃣  コードをコミット・プッシュ（Git）
🔟 GitHub Actions で自動デプロイ（CI/CD 実行）
1️⃣1️⃣ Web アプリケーションにアクセス

**各ステップに:**
- 詳しい説明

5. デプロイスクリプトが実行できない
6. Git コマンドが失敗する
7. Azure リソースが表示されない

**各問題に:**
- 症状の説明
- 診断方法
- よくある原因
- 対応手順
- ログ確認方法
